História dos computadores.

Os primeiros computadores de verdade surgiram por causa de uma guerra, a Segunda Guerra Mundial. Eles precisavam fazer contas muito difíceis e rápidas, que nenhuma pessoa conseguiria. Então, em vez de usar peças de metal como as calculadoras antigas, eles construíram máquinas gigantes usando válvulas, um tipo de lâmpada.Uma dessas máquinas foi o ENIAC.

O que é HTML?

O HTML (HyperText Markup Language) é a linguagem padrão para criar páginas da web. Ele usa "tags" para estruturar o conteúdo na web. Ela simplesmente organiza o conteúdo e diz ao navegador: "Isto é um título", "Isto é uma imagem", "Isto é um link". É o esqueleto de toda página que você visita na internet.

O que são tags?

As tags são como etiquetas que dizem ao navegador como mostrar o conteúdo. Elas são escritas entre sinais de menor e maior. Usa-se como uma tag de abertura e uma tag de fechamento.
